# Inicio

Este sitio está pensado para preparar EDNL con tres recursos:

1) **Docs** (aquí): teoría y recetas tipo examen.
2) **Labs**: Heap y Dijkstra interactivos.
3) **Recursos**: PDFs + guía HTML estática + materiales extra publicados desde `EDNL/`.

## Recomendación de uso

- Lee el tema en Docs.
- Haz el Lab correspondiente para fijar el algoritmo.
- Vuelve a los PDFs / guía HTML cuando necesites ver enunciados originales.

## Enlaces rápidos

- Labs: `/labs/heap`, `/labs/dijkstra`
- Recursos: `/recursos`
