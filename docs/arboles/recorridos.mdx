# Recorridos

En examen, lo importante es no equivocarse con el orden:

- **Preorden:** Nodo → Izq → Der
- **Inorden:** Izq → Nodo → Der
- **Postorden:** Izq → Der → Nodo
- **BFS (niveles):** por capas desde la raíz
